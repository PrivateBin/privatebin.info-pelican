(static) setCustomHandler(newHandler)

set a custom handler, which gets all notifications. This handler gets the following arguments: alertType (see array), $element, args, icon If it returns true, the own processing will be stopped so the message will not be displayed. Otherwise it will continue. As an aditional feature it can return q jQuery element, which will then be used to add the message there. Icons are not supported in that case and will be ignored. Pass 'null' to reset/delete the custom handler. Note that there is no notification when a message is supposed to get hidden.
